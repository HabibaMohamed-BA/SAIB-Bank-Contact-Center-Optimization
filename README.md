# SAIB Bank Contact Center Workforce Optimization

## 📊 Project Overview

This is an end-to-end graduation project focused on optimizing contact center operations in the banking sector using masked data.

## 🎯 Objectives

* Improve workforce planning
* Analyze agent performance
* Identify peak call hours
* Enhance operational efficiency

## 🛠 Tools Used

* Excel (Advanced Excel, Pivot Tables, Formulas)
* Power BI (Dashboard & Visualization)
* PowerPoint (Presentation)

## 📈 Key Analysis

* Call volume trends over time
* Agent performance comparison
* Peak hours identification
* Service level evaluation

## 💡 Key Insights

* Peak call times were identified during specific hours
* Performance differences were observed among agents
* Opportunities for better workforce distribution were highlighted

## 📂 Project Files

* Dashboard (PDF)
* Masked dataset (Excel)
* Project presentation (PDF)

## 🔒 Data Note

The dataset used in this project is masked to ensure privacy and confidentiality.
